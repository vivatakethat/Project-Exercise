
此程序用来维护仓库存储的零件的信息的数据库。程序围绕一个结构数组构建，且每个结构包含以下信息：零件的编号、名称以及数量。程序将支持下列操作。


1. 添加新的零件编号、名称和初始化的现货数量。
	>如果零件已经在数据库中，或者数据库一已满，那么程序必须显示出错信息。


2. 给定零件编号，显示出零件的名称和当前的现货数量。
>如果零件编号不再数据库中，那么程序必须显示出错信息。


3. 给定零件编号，改变现有的零件数量。如果零件编号不再数据库中，那么程序必须显示出错信息。


4. 显示列出数组库中全部信息的表格。
> 零件必须按照录入的顺序显示出来。


5. 终止程序的执行
		

	使用i(插入)、s(搜索)、u(更新)、p（显示）和q(退出)分别显示这些操作。
